# Hello, World!
We are the maintainers of RegPoints, a points management system for Ponytown!

This little blog is where we will post and document our work on RegPoints, share news, and more! 

Our goal is to be transparent about how we operate RegPoints.

We will not make any blog post schedule, but be sure we will talk alot in the future!

---

To celebrate this first post, we want to share some news about RegPoints:

## Version 2.0 is coming!

### General release
We are currently working on releasing regpoints **globally** meaning you will not have to use invite links anymore to create your regiment!
This is a huge step, as regpoints, even before in the past, it was only available if you had contacts with the regi that already had regpoints.

### Economy
We are also working on a economy system directly built into RegPoints, allowing you to manage points and economy on the same bot!

The economy system will be released in 2 parts: 

***The first part - Coming within the 2.0 release:***
    - The very basic: Add money, remove money, check balance, etc...
    - Possibility to add money directly from patrol creation.
    - Discord commands and management from the dashboard.

***The second part - Coming later:***
    - The shop system
    - Locking addons/item unlocking behind discord login (to be sure you belong to the regiment)
    - More features to come!

### Many bugs fixed and more to come! 

When we will release the 2.0, we will probably do a changelog in the blog to explain all the new features and new stuff!
