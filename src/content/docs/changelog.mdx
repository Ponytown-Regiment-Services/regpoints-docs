# Upcoming Features (Roadmap)

## Version 2.x

- **Last Patrol on Regiment:** Display the last patrol infos (roster, time, hoster, etc...) directly from the Discord bot.
- **Bulk point give from discord:** Allow bulk point allocation to members via Discord commands.
- **Full webhook revamp:** Overhaul the existing (and broken) webhook system.
- **Event Schedule:** Implement a schedule for upcoming events and patrols.
- **Better Feedback:** Improve the feedback system for users to report issues and suggestions.
- **Uniform locked behind discord login:** Restrict access to certain uniform features until the user logs in with Discord.
