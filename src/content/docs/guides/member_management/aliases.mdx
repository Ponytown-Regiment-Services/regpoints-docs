Aliases (or "claims") is the system we uses to let people having multiple claims in a regiment in RegPoints.
This also allows staff to not have to search for the original username of someone.

The management of aliases can be done here: 

`Member > Member List > Click a member > Member Aliases`

![The member aliases page](/imgs/aliases/memberPage.png)
> *The member management page of Iota.*

## Management of Aliases

### Add an alias
To add a new alias to a member, enter their new alias into the 'Add new alias name...' text field and click the 'Add Alias' button.

![Adding an alias to a member](/imgs/aliases/alias-add.png)
> *Adding `PTIOH` as the primary alias of Iota.*

### Add/Set an alias as primary

:::note
'Primary' alias is to label a certain alias rather than others, for exemple, it can be use to know who is playing whish role in a lore-focused regiment.
it has no other effects.
:::

:::caution
Only one alias can be set as primary at a time. You cannot delete an alias if it is set as primary, unless you ask a RegPoints dev to do it for you.
:::

To set an alias as primary, click the 'Set as Primary' button next to the alias you want to set as primary.
![Setting an alias as primary](/imgs/aliases/alias-setPrimary.png)

### Remove an alias
To remove an alias from a member, click the 'Remove' button next to the alias you want to remove.
![Removing an alias from a member](/imgs/aliases/alias-remove.png)
