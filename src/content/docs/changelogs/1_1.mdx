## Version 1.1
> First few addictions. Fixed all deprecated PHP functions since regpoints has been stopped.

### New Features

- **Added** a “ping” endpoint to check the bot status.
- **Added** a documentation part (still a work in progress).

### Bug Fixes

- Fixed various issues related to **PHP versions**.
- Fixed Emails not being sent while **Verification**.
- Fixed **redirection bugs** from register and verification to login.
- Fixed various issues related to **session management**.
- Fixed an incorrect call during **time command** on the discord bot.
- Fixed the **award.php** file not displaying properly.
- Fixed an issue with the **nominate.php** file where the nominee was not being properly validated.
- Added a **[Privacy Policy](https://alpha.regpoints.fr/privacypolicy.php)** and a **[Terms of Service](https://alpha.regpoints.fr/termsofservices.php)**.

### Bot Changes

- Added a welcome System! Still in early works, but functional.
- Fixed an API link issue where ONE SINGLE character was missing.
- Fixed an issue about time by default being unknown.
