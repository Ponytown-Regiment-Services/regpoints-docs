## Version 1.2

<Badge text="Latest version" variant="success" />

> Revemped feedback feature and 4th branches.

### Website additions

- Added A built-in [feedback](https://alpha.regpoints.fr/feedback.php) system!
- Added 4th branch to the Rank tab.

### Bot additions

- Added a welcome System! Still in early works, but functional
- Added a “ping” endpoint to check the bot status.
- Added a “last patrol” command to display the last patrol details.

### Bug Fixes

- Fixed various issues related to our hosting service (we still love you Pulse 💖).
- Fixed an issue where the bot would not respond to certain commands.
- Fixed new webhook links not being recognized. (now accepting discord.com/api/webhook as well)
